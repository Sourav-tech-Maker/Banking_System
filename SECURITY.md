# Security Policy

## Supported Versions

This project is currently under active development. Security fixes are provided for the latest version available on the `main` branch.

| Version | Supported |
| ------- | --------- |
| Latest version on `main` | ✅ |
| Older versions | ❌ |

## Reporting a Vulnerability

If you discover a security vulnerability, please report it privately through GitHub's **Private Vulnerability Reporting** feature.

Please do not create a public issue containing sensitive security details.

When submitting a report, include:

- A clear description of the issue
- The affected feature, file, route, or API endpoint
- Steps to reproduce the vulnerability
- The possible impact
- Screenshots or logs, when useful
- A suggested fix, if available

Reports will be reviewed as soon as possible. You may receive a request for additional information if the issue cannot be reproduced.

If the vulnerability is confirmed, it will be investigated and fixed in a future update. If the report is not considered a security issue, a brief explanation may be provided.

Thank you for helping improve the security of this project.
